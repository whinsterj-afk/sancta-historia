# Sancta Historia

Uma plataforma interativa para explorar a história dos santos da Igreja Católica através de uma linha do tempo dinâmica, mapa mundial interativo e correlação com os acontecimentos históricos de cada época.

## Funcionalidades previstas

- Linha do tempo de Jesus Cristo até os dias atuais
- Santos contextualizados historicamente
- Mapa interativo das jornadas dos santos
- Eventos mundiais correlacionados
- Papas e concílios
- Visualização geográfica da expansão do Cristianismo

## Tecnologias

- Next.js
- Supabase
- Vercel
- TypeScript
- Tailwind CSS
